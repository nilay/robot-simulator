# Robot Simulator
Simple ruby program to simulate to robotic move on given grid

## Prerequisite
* ruby 2.3.1

## Installtion

* Clone this repo
* `cd` to project directory
* `bundle install`
* type `ruby robo-console.rb`
* start typing command like:

```
PLACE 0,0,EAST
MOVE
REPORT
LEFT
REPORT
```
